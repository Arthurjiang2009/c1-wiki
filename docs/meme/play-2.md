# 打两个

## 由来

来源未知，起源不晚于八年级下学期。

## 杰出贡献者

wxn，每当遇到体育馆窗口期时，总会找打篮球的同学说“打两个打两个”，起到类似“我去体育馆，你去吗”的效果。

## 应用

在我班篮球集体中盛行，在其他运动类集体中也小有传播。被广泛应用于去体育馆前的匹配过程中。

通常有两种使用方式：

| 效果 | 安全性 | 使用方式                                             |
| ---- | ------ | ---------------------------------------------------- |
| ★★★★ | ★      | 在全班大吼一嗓子，一般为午休前、考试后或老师未进班前 |
| ★★   | ★★★    | 和某些同学单独谈话，一般为下课或放学                 |

在使用方式一时，有小概率会 **正巧** 碰上刘总进班，产生有较大的喜剧效果
