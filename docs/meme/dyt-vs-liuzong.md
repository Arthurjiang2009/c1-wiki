# 一身反骨/又进化了

由于这两个梗由同一个实例演化而来，为避免重复，在此一并叙述。

## 由来

在一个九年级的课间，刘总要求当周值日生dyt同学做值日，dyt同学厉声拒绝了刘总。请看画面：

```
刘总：dyt，起来做值日！

dyt:就不！

刘总（语气严厉，声音尖锐）：额，**又进化了**是吧！

```


在下一节数学课上，刘总就此事说：“额，我们班有的同学 **一身反骨**啊，老师讲的他不听对吧，啊，进化啦”（眼神瞟向dyt）。

## 应用

当有的同学拒绝另一同学时，常常会用：“进化了是吧”来表达自己的不满。而一身反骨则专指dyt。

由于该梗诞生于九年级下学期期中后，诞生时间过短，未得到充分发展。但据和其类型相同的梗（报警）来看，若该梗能顺利发展，将发展为一个具有代表性的现象级大梗。
